Panduan Penggunaan

Cara Menggunakan

  1. Input Manual:
    • Catat data PPM dan pH dari Blynk
    • Masukkan data (pisahkan dengan koma)
    • Klik "Hitung Parameter Optimal"
  2. Import CSV:
    • Export data dari Blynk format CSV
    • Pastikan ada kolom: ppm, ph
    • Upload file dan pilih minggu
    • Klik "Analisis Data CSV"
  3. Gunakan hasil sebagai setpoint di Blynk/Arduino
  4. Ulangi setiap minggu

Algoritma Decision Tree

Sistem menggunakan algoritma pohon keputusan untuk:

  • Menganalisis stabilitas parameter
  • Mendeteksi anomali dalam data
  • Memberikan rekomendasi penyesuaian

Tips Penting

  • Minggu 1: PPM rendah untuk perkembangan akar
  • Minggu 2-3: Tingkatkan PPM untuk pertumbuhan daun
  • Minggu 4: Kurangi PPM jika daun menguning
  • Jaga suhu air stabil
